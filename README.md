# Lap Time Manager

A simple C program for recording lap times, sector breakdowns, and total calculations.

## Build (MSYS2 / MinGW / GCC)
To compile the program with gcc:

```bash
gcc Lap-Time-Manager.c -o Lap-Time-Manager
```

Or use the included VS Code task (`.vscode/tasks.json`) if you are on Windows with MSYS2/ucrt64.
